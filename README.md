# TitaniumApiClient

Ruby client for interacting with Titanium.IO's JSON API

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'titanium_api_client'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install titanium_api_client
